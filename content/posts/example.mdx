# Contoh Penggunaan Code

Ini adalah inline code: `const example = "hello"` 

Dan ini adalah code block:

```javascript
const greeting = "Hello World";
console.log(greeting);
```

Atau untuk bash:

```bash
echo "Hello World"
npm install package-name
```
