```python
def calculate_sum(numbers):
    total = 0  # Initialize total
    for num in numbers:  
        total += num     
        print(f"Current sum: {total}")
    return total

numbers = [1, 2, 3, 4, 5]
result = calculate_sum(numbers)
```
