# Useful Bash Scripts for DevOps

This script monitors CPU, Memory, and Disk usage: 

```bash
#!/bin/bash
echo System Resource Monitor
echo "CPU Usage:"
top -bn1 | grep "Cpu(s)" | sed "s/.,[0−9.]∗[0−9.]∗% id./\1/" | awk '{print 100 - $1}'%
echo "Memory Usage:"
free | grep Mem | awk '{print $3/$2 100.0}'%
echo "Disk Usage:"
df -h | awk '$NF=="/"{print $5}'
